## About:
Demonstration of Skija APIs from Clojure.

## Pre-requisite:

- JDK 11
- Clojure 1.10.1
- `clojure` command-line tool 1.10.1.727 or later

## Up & Running:

From CLI run the below command:

Windows/Linux:

```
clj -M -m lwjgl.main
```

macOS:

```
clj -J-XstartOnFirstThread -M -m lwjgl.main
```
